# 封膜庫存監控規則記憶

## 固定網址

建議網址路徑：

`http://localhost/phoenixes-film-inventory/`

同網段分享網址：

`http://192.168.5.47/phoenixes-film-inventory/`

或：

`http://Phoenixes-MD/phoenixes-film-inventory/`

命名理由：

- `phoenixes`：公司識別。
- `film`：封膜品項。
- `inventory`：庫存監控用途。

若未來要讓同事用更短網址，例如 `http://phoenixes-film-inventory/`，需要透過公司內網 DNS 或 Windows hosts 設定。

穩定性注意：

- 目前是由這台電腦提供內網網頁。
- 這台電腦關機、睡眠、斷網，或 Python 網頁服務停止時，同事就看不到。
- 若要長期穩定，建議之後改成 Windows 開機自動啟動或正式內網服務。

## 專案位置

所有相關檔案統一放在：

`D:\封王封膜庫存監控`

主要檔案：

- `public\phoenixes-film-inventory\index.html`：網頁畫面。
- `public\phoenixes-film-inventory\dashboard-data.js`：目前解析後的庫存資料。
- `scripts\build_dashboard.py`：讀取 ERP Excel 並產生資料檔。
- `scripts\start_dashboard.ps1`：更新資料並啟動本機網頁。
- `docs\分類與庫存規則.md`：分類、顏色與維護規則記憶。

## 庫存顏色規則

所有品項：

- 20 卷以下：紅色

130 寬度品項：

- 低於 60 卷：橘色
- 60 卷以上：綠色

160 / 180 / 225 寬度品項：

- 低於 40 卷：橘色
- 40 卷以上：綠色

三櫻系列全部品項：

- 低於 40 卷：橘色
- 40 卷以上：綠色
